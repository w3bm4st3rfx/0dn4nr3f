What is React Native CLI?

React Native CLI is a great solution if you want to integrate some native code (written in Java/Kotlin or Swift/Objective-C) into your project or even write your own.

In recent years, React Native has had many improvements in its infrastructure, making your App as similar to Native as possible. With React Native it is possible to integrate many native mobile functionality like camera, file manager, Bluetooth and more thanks to the great work and support of the huge community behind React Native.

However, React Native CLI has some crucial limitations that must be considered. One of these limitations is that, unlike Expo, you need to have the right equipment to compile your app to .apk or .ipa, to send your App to production on the two main mobile platforms (iOS and Android).

You need a Mac and Xcode for the Apple Store, and an Android studio for the Play Store. Also, when you're ready to ship your App to production, you'll need to handle all the configurations, signatures, certificates, and many other things.
